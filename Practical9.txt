9. Create Function

i) to find gcd of two numbers.

SQL> SET SERVEROUTPUT ON;

SQL> CREATE OR REPLACE FUNCTION gcd(a NUMBER, b NUMBER)
RETURN NUMBER IS
BEGIN
  IF b = 0 THEN
    RETURN a;
  ELSE
    RETURN gcd(b, MOD(a, b));
  END IF;
END;
/

SQL> SELECT gcd(8, 12) from dual;
